# chapter1 导论

“计算机科学中的任何问题都可以通过增加一个中间层来解决。” ———David Wheeler

计算机系统的抽象层次通过分层组织，底层由硬件实现，高层由软件实现。虚拟机则涉及硬件和软件分界出及其附近的抽象层次，在这些层次上软件从运行它的机器中分离出来。

虚拟机的研究学习主要包括三个方面：体系结构、操作系统、编程语言。如果要研究透虚拟化技术，学习这三方面的内容是重中之重。
